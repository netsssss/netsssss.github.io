css编入集

  • Author NetsSsSs
  • Date Aug 18
  • Categories no_categories

1. 盒子模型

  1. 每一个元素都会带有宽高属性，即使高度为0

  2. 元素默认的margin，padding，border值为0

  3. margin可以设置负值

    1. 当块元素没有设定宽度时，负left和right会增加左边或右边的宽度
    2. 设置负top时会向上移动
    3. 设置负bottom时会向上移动并减少元素的css读取高度(缩减了元素的content区域)

  4. box-sizing拥有两个常用属性

    1. content-box(默认): 宽度不包括border和padding
    2. border-box(IE默认): 宽度包括border和padding

ie和chrome显示图片清晰度不一致的解决办法

  • Author NetsSsSs
  • Date Nov 10
  • Categories no_categories

变得同样清晰

image-rendering

出现这种情况主要是因为 Chrome 默认对图片做了反锯齿。而IE没做，所以感觉上IE出现了“更清晰”感觉。

给图片css加上 image-rendering: pixelated; 就可以同样清晰了。
