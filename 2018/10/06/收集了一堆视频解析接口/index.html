收集了一堆视频解析接口...
发布于：2018-10-06 23:26:37 标签：/ api / 访问：

起因：
无意中看到很久之前做的静态网页，感慨了半天当时的自己辛辛苦苦往网页里填满了静态数据还以为网页就是这么做的…

想法：
静态网页是个视频网页，我就想要不就用这个干脆做个视频解析网站吧。于是开始到网上搜视频解析网站看看，发现他们解析出视频之后都把视频塞到了各种乱七八糟的播放器里面，风格太不统一了，就想着干脆抓他们的源地址出来自己加统一的播放器。
一开始以为解析接口是靠给请求加cookie和token之类的获得源视频，只要拿到这些东西就可以自己做解析网站了…结果人家果然技高一筹，解析放在了后台，前端直接请求后台的api就可以拿到源地址了，后来我就想，既然他的后台返回的是源地址，那我就直接伪造他前台的请求好了拿他解析好的源地址好了，于是就有了下面的代码
